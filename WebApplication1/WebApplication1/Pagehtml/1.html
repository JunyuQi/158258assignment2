Quick Sort

Problem Statement

Given you a length of n series of integer numbers.
Please use Quick Sort to sort this series from smallest to largest.
And output the ordered series in order.

Input

The input consists of two lines, the first of which contains the integer n.
The second line contains n integers (all integers are between 1 to 109.) representing the entire series.

Output

The output is a single row containing n integers representing sequences of numbers.

Sample Input

            5

            3 1 2 4 5

Sample Output

            1 2 3 4 5
