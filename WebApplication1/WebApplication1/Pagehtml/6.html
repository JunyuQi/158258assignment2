Linear Dynamic Programming

Problem Statement

Given a number triangle as shown in the figure below, starting from the top, at each node, you can choose to move to the node at the bottom left or to the node at the bottom right, until you go to the bottom, and ask to find a path so that the sum of the numbers on the path is the largest.
7
3 8
8 1 0
2 7 4 4
4 5 2 6 5

Input

The first line contains the integer n , representing the number of layers of the number triangle.
Next n rows, each containing a number of integers, where i line represents the number triangle i integer that the layer contains.

Output

Outputs an integer representing the sum of the largest path digits.

Sample Input

            5

            7

            3 8

            8 1 0 

            2 7 4 4

            4 5 2 6 5

Sample Output

            30
