Binary Search

Problem Statement

Given a length of n in ascending order an array of integers, and q queries.
For each query, an element k is returned The start and end positions (positions from 0 Start counting).
If the element is not present in the array, -1 -1 is returned.

Input

The first line contains the integer n and q to indicate the length of the number and the number of queries.
The second line contains n integers (all between 1 and 10000 range), representing a complete array.
Next q rows, each containing an integer k , representing an interrogation element.

Output

Total q Rows, each containing two integers, representing the start and end positions of the sought element.
If the element is not present in the array, -1 -1 is returned.

Sample Input

            6 3

            1 2 2 3 3 4

            3

            4

            5

Sample Output

            3 4

            5 5

            -1 -1
