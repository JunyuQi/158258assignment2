Select Points

Problem Statement

Given N Closed interval [ai,bi] Please choose as few points as possible on the number line, so that each interval contains at least one selected point. Outputs the minimum number of selected points.
Points that are located at the end of the interval are also counted as within the interval.

Input

The first row contains the integer N to indicate the number of intervals.
Next N Rows, each containing two integers ai, bi , representing the two endpoints of an interval.

Output

Outputs an integer representing the minimum number of points required.

Sample Input

            3

            -1 1

            2 4

            3 5

Sample Output

            2
